# React Native :app:installDebug FAILED

Build React native error :app:installDebug FAILED

## Android

### Windows
1. `cd android`
2. `./gradlew clean`
3. `cd ..`
4. `npm cache clean --force`

### MacOS
1. `cd android`
2. `./gradlew clean`
3. `cd ..`

**Then**
`reboot your android device`
